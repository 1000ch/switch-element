# [switch-element](http://1000ch.github.io/switch-element)

## About

Switch UI Element like iOS.

## Usage

Install with **npm** .

```bash
$ npm install switch-element
```

Or from **bower** .

```bash
$ bower install switch-element
```

Put `<switch-element>` tag.

```html
<switch-element></switch-element>
```
